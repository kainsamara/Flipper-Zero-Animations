## How to install
1) Unzip L1_Garage_128x64 folder in sdcard/dolphin/
2) Add to sdcard/dolphin/manifest.txt this code (Sample, change &ANIMATION_NAME to name of animation):
```
Name: &ANIMATION_NAME
Min butthurt: 0
Max butthurt: 5
Min level: 1
Max level: 3
Weight: 3
```
3) Reboot your Flipper Zero
